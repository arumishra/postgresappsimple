# PostgreSQL CRUD App

A simple Python CRUD application using PostgreSQL and psycopg2.

## Features
- Create database table
- Insert one record
- Insert multiple records
- View records
- Delete records

## Setup

Clone the repository:

```bash
git clone <repo-url>

Install dependencies:

pip install -r requirements.txt

Create a .env file:

DB_HOST=localhost
DB_NAME=your_database
DB_USER=your_user
DB_PASSWORD=your_password
DB_PORT=5432

Run:

python app.py
